# Security Policy

## Supported Versions

We currently support the latest version of Spec Kit Extensions.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities by opening an issue with the "security" label, or if it is sensitive, please contact the maintainers directly through their GitHub profiles.
